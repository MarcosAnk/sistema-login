# Sistema de Login

Sistema de autenticação funcional desenvolvido com Node.js e Express.

## Tecnologias

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express
- **Segurança:** CORS

## Como executar

```bash
# Clone o repositório
git clone https://github.com/MarcosAnk/sistema-login.git

# Entre na pasta backend
cd sistema-login/backend

# Instale as dependências
npm install

# Inicie o servidor
npm start
